## Coding style⌨️
### C/C++
```C
#include <stdio.h>

int main() {
  for (int i = 1; i <= 9; ++i) {
    for (int j = 1; j <= 9; ++j) {
      printf("%d * %d = %d \n", i, j, i * j);
    }
  printf("\n");
  }
}
```
### Python
```Python
for i in range(1, 10):
  for j in range(1, 10):
    print(i, '*', j, '=', i*j)
  print()
```
